**Resumen**
- **Proyecto**: Manos_Locales — aplicación Android con Jetpack Compose, Hilt, Retrofit y patrones MVVM.
- **Propósito**: Cursada de materia Tecnologías móviles / catálogo de productos con autenticación, favoritos y carrito de compras.
- **Lenguaje / Herramientas**: Kotlin, Jetpack Compose, Hilt (DI), Retrofit (HTTP), Coroutines, DataStore/SharedPreferences, Material3.
- **Autores**: Carlos Emanuel Gomez Boneli y Agustin Cardetti Facchin.
- **Universidad evaluadora**: UNDEF – CRU IUA (Instituto Universitario Aeronáutico)
- **Docente evaluador**: Ingeniero Nicolas Fanin
